## General info
The goal of this task was to prepare a simple command line interface which provides the following commands:

-- version

--help

<index.html>

The last command starts the HTTP server which serves the provided HTML file.
	
## About the code

"fmt"
"os"
"regexp"
"net/http"
"bufio"
"log"
